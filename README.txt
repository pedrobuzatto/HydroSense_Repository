Antes de começar a fazer o código eu fiz as seguintes coisas:
    Instalei o node.js no meu computador com o link https://nodejs.org/en/download/prebuilt-installer

    Importei a biblioteca do websocket com dois comandos no terminal do VS Code:
    npm install socket.io --legacy-peer-deps
    npm install socket.io-client --legacy-peer-deps

Feito isso, vários arquivos apareceram na pasta do HydroSense. Como eles já estão aqui, acho que só
instalar o node.js vai bastar para rodar os códigos
